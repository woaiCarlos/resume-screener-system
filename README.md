# 基于 Python 的简历智能筛选系统

> 上海开放大学本科毕业设计（论文）

## 项目简介

本系统旨在利用 Python 技术实现简历的智能解析、筛选与可视化展示，帮助招聘人员快速从大量简历中筛选出符合条件的候选人。

## 功能模块

| 模块 | 说明 |
|------|------|
| 简历数据解析模块 | 解析 PDF、Word 等格式简历，提取关键字段（姓名、学历、技能、工作经历等） |
| 关键词匹配与评分模块 | 构建职位需求关键词词库，采用 TF-IDF 算法进行匹配度打分 |
| 人才可视化展示模块 | 基于 ECharts 实现雷达图、柱状图等可视化展示 |
| Web 管理界面 | 基于 Django 框架搭建系统后台 |

## 技术栈

- **语言**: Python
- **后端框架**: Django
- **数据处理**: TF-IDF、NLP
- **前端可视化**: ECharts
- **简历解析**: PDF、Word 文本提取
